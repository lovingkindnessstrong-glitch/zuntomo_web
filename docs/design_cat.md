# ページ詳細設計書: 猫おどり部（Neko-Odori Club）

## 1. ページ基本情報
* **ページタイトル:** 猫おどり部 | ZUNTOMO
* **URLパス:** `/activities/neko-odori/index.astro`
* **目的:** 函南町の伝統ある奇祭「かんなみ猫おどり」の継承と普及を目指し、一緒にステージを盛り上げる幅広い世代の「にゃんこ（猫の踊り手）」を募集する。

## 2. ヒーローセクション（最上部）
* **メインキャッチコピー:** 
    世代を超えて「にゃんこ」になろう！
    2万人が熱狂する奇祭・かんなみ猫おどりを一緒に盛り上げませんか？
* **ビジュアル指示:** 猫メイクやコスプレで楽しく踊る様子や、お祭りの活気、可愛らしさが伝わるメインビジュアルを配置する。

## 3. コンセプト・ストーリー（ZUNTOMO猫おどり部の想い）
* **見出し:** 函南町で30年以上愛される奇祭を、みんなで継承！
* **本文要素:**
    * 10匹の保護猫と暮らす「猫大好きインストラクター」が、かんなみ猫おどり実行委員会様と二人三脚でダンスの練習・普及活動を行っています。
    * 幼稚園のお子様から、アイドル猫、パパママ猫、そして70代のシニア世代まで、幅広い年齢層の踊り手が大活躍中です。
    * 毎年2万人の来場者を誇る地域の伝統ある楽しいお祭りを、最高の仲間たちと一緒に盛り上げましょう！

## 4. クラス・イベント詳細（スケジュールと参加案内）
* **見出し:** 練習会 ＆ イベント出演 詳細

* **カード1: ズンとも猫おどり部😺無料練習会**
    * **開催日時:** 毎週水曜日 18:45〜19:15（30分）
    * **会場:** 函南町西部コミュニティセンター
    * **特記事項（重要）:** ※毎年7月・8月のみ、会場が「函南町文化センター」に変更となります。
    * **料金:** 0円（無料）
    * **タグ:** `無料` `初心者歓迎` `全世代参加OK` `かんなみ猫おどり`

* **カード2: 発表会 ＆ 本番ステージ**
    * **イベント（月1回）:** 月に1回程度、地域のイベント等でダンスの発表があります（自由参加）。
    * **本番ステージ（10月）:** 毎年10月に開催される本番「かんなみ猫おどり」の巨大ステージ（来場者数2万人）に出演します！
    * **衣装について:** 普段の練習着は自由です。本番は思いっきりの「猫コスプレ＆猫メイク」で変身して楽しみましょう！
    * **タグ:** `自由参加` `猫コスプレ必須（本番）` `思い出作り`

## 5. デザインとレイアウトの指示（Tailwind CSS）
* `Layout.astro` でページ全体をラップすること。
* 大人クラスやゴールドクラスのデザインを踏襲しつつ、猫の可愛らしさやお祭りの楽しさが伝わるよう、イエローやオレンジなどのポップで元気なアクセントカラーを使用すること。
* スケジュール部分は既存のカードデザインを踏襲し、2つのカードを並べて表示すること。
* 文字色は見やすく `text-gray-800` などを指定し、背景と同化しないよう徹底すること。
* 「7月・8月の会場変更」に関する注意事項は、サルサクラスの休講情報と同様に目立つようにスタイリングすること。
