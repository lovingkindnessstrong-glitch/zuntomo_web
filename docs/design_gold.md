# ページ詳細設計書: ゴールドクラス（S-DANCE GOLD）

## 1. ページ基本情報
* **ページタイトル:** ゴールドクラス（S-DANCE GOLD） | ZUNTOMO
* **URLパス:** `/activities/gold/index.astro`
* **目的:** シニア層、ハンディキャップを持つ方、リハビリ目的の方に、音楽と笑顔にあふれた安全な運動の場を提供していることを伝える。

## 2. ヒーローセクション（最上部）
* **キャッチコピー:** めちゃめちゃ楽しく運動しよう！
* **サブコピー:** 誰もが知っているあの曲で。心も体も、そして脳もイキイキ若返る特別な60分。
* **ビジュアル指示:** 参加者全員がバランスボールに座って笑顔で手を挙げている集合写真（`IMG_20250713_143408.jpg`）をヒーロー背景、またはメインビジュアルとして大きく配置する。

## 3. コンセプト・誕生ストーリー（S-DANCE GOLDの想い）
* **見出し:** 音楽と笑顔で、誰も置き去りにしないフィットネスを。
* **本文要素:**
    * インストラクター自身の半身不随（脳出血）の経験から、「シニアやハンディキャップがある方でも心から楽しめる運動指導をしたい」という強い想いでスタートしました。
    * 日本の皆様に本当に心から楽しんでいただくため、みんなが知っていて口ずさめる流行曲やポップスを中心に、独自のプログラム「S-DANCE GOLD」を開発。
    * 運動だけでなく、「シナプソロジー®️（脳トレ）」や「バランスボール」を取り入れ、心身機能の維持・向上と脳の活性化を同時に図ります。

## 4. 対象者・こんな方におすすめ
* **見出し:** どなたでも参加可能！あなたのペースで楽しめます
* **リスト項目:**
    * 健康寿命を延ばしたいアクティブシニアの方
    * 心身にハンディキャップのある方（車椅子でのご参加も大歓迎です！）
    * 医師から運動を勧められている方、または久しぶりに運動する方
    * リハビリの延長として楽しく体を動かしたい方
* **タグ（バッジ表示）:** `S-Dance™GOLD` `シナプソロジー®️` `脳トレ` `リハビリ` `バランスボール` `車椅子OK`

## 5. スケジュールと参加案内
* **見出し:** S-Dance™ GOLD クラス詳細
* **カードレイアウト要素:**
    * **開催日時:** 日曜日 13:30〜14:30（60分） ※月1回・不定期開催
    * **会場:** 沼津市総合体育館 2F 多目的スタジオ
    * **料金:** 500円（現金都度払い）
    * **参加方法:** 事前予約不要。13時のドアオープン以降、直接現地にお越しください。
* **特記事項:** 障害者サポート団体様などからの出張レッスン依頼も承っております。お気軽にご相談ください。

## 6. デザインとレイアウトの指示（Tailwind CSS）
* `Layout.astro` でページ全体をラップすること。
* 配色は、ポスター画像（`Sダンス掲示用ポスターのコピー - 1.png`）からインスピレーションを得て、ブルーを基調としつつ、イエローやレッドをアクセントにした「明るく元気でポジティブ」なトーンにすること。
* スケジュール部分は大人クラス（STANDARD）で作成したカードデザインを踏襲し、サイト全体の統一感を持たせること。
